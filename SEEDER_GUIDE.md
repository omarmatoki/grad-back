### 🎓 المشاركون (Participants)

**أمثلة بحسابات تلقائية:**

| الاسم | المشروع | الحضور | معدل الحضور | التحسن |
|------|---------|--------|-------------|--------|
| عبدالله محمد | تمكين الشباب | 8/10 | 80% | 35.4% |
| نورة أحمد | تمكين الشباب | 9/10 | 90% | 31.4% |
| منى الزهراني | الأسر المنتجة | 7/8 | 87.5% | 41.8% |
| هدى العمري | الأسر المنتجة | 8/8 | 100% | 41.7% |

**القيم المحسوبة تلقائياً:**
- `attendanceRate` = (attendanceSessions / totalSessions) × 100
- `improvementPercentage` = ((postScore - preScore) / preScore) × 100

### 📈 السجل التاريخي (Indicator History)

كل مؤشر له 3 قياسات تاريخية تُظهر التطور عبر الوقت.

**مثال - عدد المتدربين:**
- أكتوبر 2025: 650 ← زيادة 150 عن السابق (+30%)
- ديسمبر 2025: 750 ← زيادة 100 (+15.38%)
- يناير 2026: 850 ← زيادة 100 (+13.33%)

**الحسابات التلقائية:**
- `changeAmount` = recordedValue - previousValue
- `changePercentage` = (changeAmount / previousValue) × 100
- `trend` يُحسب من آخر 5 قياسات

## 🧪 اختبار البيانات المُضافة

### 1. تسجيل الدخول:
```bash
POST /auth/login
{
  "email": "admin@example.com",
  "password": "Test123456!"
}
```

### 2. جلب المشاريع:
```bash
GET /projects
# يجب أن تحصل على 3 مشاريع
```

### 3. جلب مستفيدي مشروع:
```bash
GET /beneficiaries/project/{project_id}
# يجب أن تحصل على مستفيدين المشروع
```

### 4. جلب إحصائيات مشاركين:
```bash
GET /participants/project/{project_id}/stats
# إحصائيات كاملة مع الحسابات التلقائية
```

### 5. جلب المؤشرات:
```bash
GET /indicators/project/{project_id}
# يجب أن تحصل على جميع المؤشرات
```

### 6. جلب السجل التاريخي:
```bash
GET /indicators/{indicator_id}/history
# 3 قياسات تاريخية لكل مؤشر
```
